Project Portofolio

Whistleblowing System Jepara

WBS Jepara adalah web pelaporan dilingkup ASN Pemda Jepara. web ini adalah project yang pertama kali saya buat secara fullstack dimana saat itu saya juga baru pertama kali mengenal CodeIgniter. Project ini dibuat pada Februari 2020 ketika saya masih duduk dibangku kuliah saat Magang / PKL di Diskominfo Kabupaten Jepara. web ini memiliki beberapa fitur seperti tulis aduan, mendapatkan nomor aduan setelah tulis aduan (nomor aduan digunakan untuk mengecek status aduan / pelaporan), mendapatkan info melalui email setiap status aduan berubah, cetak aduan dalam 6 bulan terakhir (admin), filter aduan, dan lain-lain. Sekarang web ini bisa dikunjungi dengan link berikut http://wbs.jepara.go.id/

Framework :
  • CodeIgniter 3
  • Bootstraps 4
